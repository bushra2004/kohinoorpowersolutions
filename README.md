# Kohinoor Power Solutions - Customer Management System

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Flask](https://img.shields.io/badge/Flask-2.3.3-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

A professional, local customer management system for Kohinoor Power Solutions - Automobile Battery and CCTV Solutions provider in Gulbarga.

## 🚀 Features
- ✅ **Local Data Storage** - SQLite database (unlimited storage)
- ✅ **Excel Export** - Download all data as formatted Excel files
- ✅ **Professional UI** - Branded with company logo and colors
- ✅ **User Authentication** - Secure admin login
- ✅ **Search & Filter** - Find customers quickly
- ✅ **Completely Offline** - No internet required
- ✅ **Responsive Design** - Works on all devices

## 🏢 About Kohinoor Power Solutions
**Location:** Darga Cross, Gulbarga (Kalaburagi)  
**Services:** Automobile Batteries, CCTV Dealers, CCTV Installation Services

```bash
